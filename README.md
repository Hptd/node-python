[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)

**简易中文节点 Python 编辑器**（NodePy）是一款面向非程序员的图形化编程工具，通过拖拽节点和连接线即可构建可执行的数据流图。支持自定义 Python 节点、AI 代码生成、多线程并发处理等高级功能。

## ✨ 功能特性

### 🎨 可视化节点编辑

| 功能 | 说明 |
|------|------|
| **多种添加方式** | 拖拽/双击/右键菜单/空格搜索 |
| **智能拼音搜索** | 支持中文拼音首字母匹配（如"fds"→"浮点数"） |
| **节点连接** | 输出端口 (橙色) → 输入端口 (蓝色) |
| **框选多选** | 左键框选，Ctrl+ 点击增选 |
| **视图操作** | 中键平移，滚轮缩放 (1.15 倍率) |
| **节点组** | 多节点打组、嵌套、组内编辑 |

### 📦 节点库系统

**内置节点分类**：
- **输出类**：打印、日志
- **常量类**：布尔/整数/浮点/字符串/列表/字典
- **数据处理**：数据提取、类型检测
- **循环控制**：区间循环、List 循环、多线程处理
- **文本处理**：正则提取（含智能生成器）
- **Debug**：调试输出、执行时间

**扩展能力**：
- ✅ 自定义 Python 节点（内置代码编辑器）
- ✅ AI 辅助生成节点代码
- ✅ 节点插件导入/导出（JSON 格式）

### ⚙️ 参数输入系统

根据参数类型自动匹配输入控件：

| Python 类型 | UI 控件 | 输入格式 |
|-------------|---------|----------|
| `bool` | 复选框 | ✓/✗ |
| `int` | 整数输入框 | `42` |
| `float` | 小数输入框 | `3.14` |
| `str` | 文本输入框 | `Hello` |
| `list` / `dict` | JSON 编辑器 | `["a", "b"]` / `{"k": "v"}` |
| 其他 | 文本输入框 | 自动转换 |

**优先级规则**：外部连接值 > 属性面板参数值

### 🔄 循环节点

#### 区间循环 (Range Loop)
```
参数：最小值=0, 最大值=5, 步长=1
迭代序列：0 → 1 → 2 → 3 → 4
```

#### List 循环
```
输入：["苹果", "香蕉", "橙子"]
迭代：逐个处理每个元素
```

#### 多线程处理
```
输入列表：[URL1, URL2, ..., URL10]
线程数：4
返回顺序：按输入顺序 / 按完成顺序
适用场景：I/O 密集型（网络请求、文件读写）
```

### 🎯 数据提取

**路径语法示例**：

```python
数据源：{"user": {"profile": {"name": "张三", "hobbies": ["阅读", "游泳"]}}}

提取路径          结果
─────────────────────────────
user              {"profile": {...}}
user.profile      {"name": "张三", ...}
user.profile.name "张三"
user.profile.hobbies.0    "阅读"
user.profile.hobbies[1]   "游泳"
```

### 🧩 正则提取

**智能生成器工作流程**：

```
1. 输入示例文本：
   CG 渲染_2026-02-26_21-11-27
   彩虹小马_2026-02-26_21-18-55

2. 标记模板：[CG 渲染]_2026-02-26_21-11-27
            ↑ 方括号内为提取内容

3. 自动生成正则：([^_]+)_\d{4}-\d{2}-\d{2}_\d{2}-\d{2}-\d{2}

4. 测试结果 → 确认使用
```

**内置模板**：邮箱、手机号、日期、URL、文件名、IP 地址等

## 📥 安装与配置

### 环境要求

| 组件 | 版本要求 |
|------|----------|
| Python | 3.10+ |
| PySide6 | ≥ 6.5.0 |
| 可选依赖 | pypinyin, requests, pandas, numpy, openpyxl, Pillow |

### 安装步骤

```bash
# 1. 创建虚拟环境（推荐）
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行应用
python main.py
```

### 嵌入式环境配置

**为什么需要嵌入式环境？**
- 隔离用户自定义节点的依赖
- 避免污染主 Python 环境
- 支持第三方包动态安装

**配置方法**：
```bash
# 方法一：应用内初始化
点击工具栏 → 🔧 初始化环境

# 方法二：命令行配置
python -m utils.setup_embedded_python install

# 方法三：安装第三方包
点击工具栏 → 📦 依赖管理 → 输入包名 → 安装
```


### 循环节点高级用法

#### 嵌套循环
```
[外层循环] → [内层循环] → [处理节点]
    ↓           ↓
  汇总 1       汇总 2
```

#### 条件终止（需配合数据流设计）
```
[循环] → [判断节点] → [提前返回逻辑]
```

### 多线程最佳实践

**适用场景**：
- ✅ 网络请求（API 调用、网页爬取）
- ✅ 文件 I/O（批量读写）
- ✅ 数据库查询

**不适用场景**：
- ❌ CPU 密集型计算（使用批量执行更优）
- ❌ 有状态共享的操作（需额外同步）

**性能参考**：
```
任务类型：下载 10 个文件（每个约 1MB）
单线程：  ~30 秒
4 线程：   ~8 秒
8 线程：   ~5 秒
```

### 调试技巧

1. **打印中间结果**：在关键节点后连接【打印】节点
2. **类型检查**：使用【数据类型检测】节点验证输出
3. **执行时间**：启用设置中的"显示执行时间"
4. **错误定位**：控制台红色输出为错误信息，包含节点名称

## 👨‍💻 开发者指南

### 添加内置节点

**步骤 1**：在 `core/nodes/base_nodes.py` 中定义函数

```python
def 向量点积 (vec1: list, vec2: list) -> float:
    """
    计算两个向量的点积（内积）。
    
    参数:
        vec1: 第一个向量（列表）
        vec2: 第二个向量（列表）
    
    返回:
        点积结果（浮点数）
    
    异常:
        ValueError: 当两个向量长度不一致时
    """
    if len(vec1) != len(vec2):
        raise ValueError(f"向量长度不一致：{len(vec1)} vs {len(vec2)}")
    
    result = sum(a * b for a, b in zip(vec1, vec2))
    return float(result)

# 保存源代码（用于序列化）
向量点积._source = '''def 向量点积 (vec1: list, vec2: list) -> float:
    """计算两个向量的点积（内积）。"""
    if len(vec1) != len(vec2):
        raise ValueError(f"向量长度不一致：{len(vec1)} vs {len(vec2)}")
    result = sum(a * b for a, b in zip(vec1, vec2))
    return float(result)
'''
```

## ❓ 常见问题

### Q1: 自定义节点无法执行？

**检查清单**：
1. 是否已初始化嵌入式环境（🔧 按钮）
2. 节点代码是否有语法错误
3. 是否定义了正确的函数签名
4. 返回类型是否匹配

### Q2: 循环节点执行缓慢？

**优化建议**：
- 减少单次迭代的数据量
- 使用多线程处理节点（I/O 密集型）
- 避免在循环内执行耗时操作

### Q3: 如何调试数据流？

**方法**：
1. 在关键节点后连接【打印】节点
2. 使用【数据类型检测】验证类型
3. 分段执行：先执行部分节点，验证中间结果

### Q4: 第三方包安装失败？

**解决步骤**：
```bash
# 1. 确认嵌入式环境路径
python -m utils.setup_embedded_python check

# 2. 手动安装包
<path-to>/python_embeded/python.exe -m pip install <package-name>

# 3. 重启应用
```

### Q5: 画布缩放异常？

**重置方法**：
- 双击画布空白处重置视图
- 检查设置中的 `graphics.zoom_speed`（推荐 1.1-1.2）
