# 简易中文节点 Python 编辑器

基于 PySide6 的图形化节点编辑器，允许用户通过拖拽和连接节点来创建可执行的数据流图。支持自定义 Python 节点函数、持久化存储和实时执行。

## 功能特性

### 可视化节点编辑
- **拖拽/双击/右键菜单**：三种方式添加节点到画布
- **智能搜索**：支持中文拼音首字母模糊匹配（如"fds"→"浮点数"）
- **连接节点**：从输出端口（橙色）拖拽到输入端口（蓝色）
- **框选多选**：左键拖动框选，Ctrl+ 点击增选/取消
- **视图操作**：中键平移，滚轮缩放

### 节点库管理
- **分类管理**：节点按功能分类（输出、常量、提取、循环、Debug 等）
- **自定义节点**：内置 Python 编辑器动态创建节点
- **AI 生成节点**：一键生成代码模板，复制粘贴即可创建
- **节点插件**：导出/导入 JSON 格式插件包，实现模块化复用

### 参数输入系统
- **属性面板编辑**：选中节点后在右侧面板输入参数
- **智能控件匹配**：根据参数类型自动生成控件
  - `bool` → 复选框 | `int` → 整数输入 | `float` → 小数输入
  - `list/dict` → JSON 输入框 | 其他 → 文本输入
- **输入优先级**：外部连接值 > 属性面板参数值

### 循环节点
- **区间循环（Range Loop）**：按整数范围和步长迭代
- **List 循环**：遍历列表中的每个元素
- **多线程处理**：线程池并发处理列表（I/O 密集型）
- **连接模式**：
  - 连接"迭代值"：每次迭代执行的核心任务
  - 连接"汇总结果"：循环完成后执行一次

### 数据提取
- **数据提取节点**：从字典/JSON 中提取路径字段，支持嵌套访问和数组索引
  - 路径语法：`user.name`、`items[0]`、`data.list.0.id`
- **正则提取节点**：从文本中提取正则匹配内容
  - **智能生成器**：输入示例文本，用 `[]` 标记提取内容，自动生成正则
  - **常用模板**：邮箱、手机号、日期、文件名等一键使用

### 节点组（Group）
- **创建节点组**：选中多个节点 → 右键"创建组"（Ctrl+G）
- **组内编辑**：双击组进入编辑模式
- **组重命名**：双击组标题重命名

### 主题切换
- **黑白双主题**：暗黑/明亮模式一键切换
- **自动保存**：主题设置持久化

### 持久化存储
- **图表存储**：JSON 格式保存节点、连接、参数值
- **自定义节点**：自动保存到 `.node-python/custom_nodes.json`
- **应用设置**：窗口状态、主题、执行选项等

## 安装运行

### 环境要求
- Python 3.10+
- PySide6 >= 6.5.0

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行应用
```bash
python main.py
```

### 初始化嵌入式环境（可选）
首次使用或需要重置环境时：

### 添加节点

**方法一：拖拽** - 从节点库拖拽节点到画布  
**方法二：双击** - 双击节点库中的节点名称  
**方法三：右键菜单** - 画布空白处右键（或按 Space），通过分类菜单或搜索添加

### 节点操作

| 操作 | 方法 |
|------|------|
| **移动节点** | 按住左键拖动 |
| **连接节点** | 从输出端口（橙色）拖拽到输入端口（蓝色） |
| **删除连接** | 双击输入端口 |
| **删除节点** | 选中后按 Delete 或 Backspace |
| **复制节点** | 选中后按 Ctrl+D 或右键"复制并粘贴" |
| **多选节点** | 左键框选 或 Ctrl+ 点击 |
| **创建节点组** | 选中多个节点 → Ctrl+G |
| **组内编辑** | 双击节点组 |

### 参数输入

1. 单击选中节点
2. 在右侧"📥 参数输入"区域输入参数值
3. 执行时优先使用连接值，无连接时使用参数值

### 保存和加载图表

**保存图表**：点击"💾 保存为 JSON"→ 选择路径和文件名 → 保存

**覆盖加载**：点击"📂 加载 JSON"→ 选择文件 → 清空当前画布并加载新图表

**增量加载**：点击"📂 加载 JSON"下拉箭头 → 选择"增量加载"→ 保留当前内容并添加新图表

### 执行流程图

1. 确保节点已正确连接
2. 设置各节点的参数值（如需要）
3. 点击"▶ 运行"按钮
4. 执行结果显示在底部控制台
5. 如需停止，点击"⏹ 停止"按钮

### 循环节点使用

#### 区间循环（Range Loop）
1. 添加"区间循环"节点
2. 设置参数：最小值、最大值、步长
3. 连接"迭代值"到处理节点
4. 连接"汇总结果"到接收完整结果的节点

**示例**：最小值=0, 最大值=5, 步长=1 → 迭代值：0, 1, 2, 3, 4

#### List 循环
1. 添加"List 循环"节点
2. 输入列表数据（JSON 格式，如 `["a","b","c"]`）
3. 连接"迭代值"到处理节点
4. 连接"汇总结果"到接收完整结果的节点

#### 外部输入控制
循环节点支持通过外部节点输入控制参数：
- 区间循环：连接整数输出到"最小值"、"最大值"、"步长"端口
- List 循环：连接列表输出到"列表数据"端口

### 数据提取节点

**用途**：从结构化数据（字典/JSON）中提取指定路径的字段值

**使用步骤**：
1. 添加"数据提取"节点
2. 连接数据源到 `data` 端口（或在属性面板输入 JSON）
3. 在 `path` 参数输入提取路径

**路径语法**：
| 语法 | 说明 | 示例 |
|------|------|------|
| `字段名` | 访问字典的键 | `name` |
| `字段 1. 字段 2` | 嵌套访问 | `user.name` |
| `数组.索引` | 访问数组元素 | `items.0` |
| `数组 [索引]` | 方括号格式 | `items[0]` |
| 组合使用 | 混合嵌套 | `data.items[0].name` |

**示例**：
```
输入数据：{"user": {"name": "张三", "age": 25}}
提取路径：user.name
输出结果：张三
```

### 正则提取节点

**用途**：从文本中提取符合正则表达式模式的内容

**使用步骤**：
1. 添加"正则提取"节点
2. 连接输入文本到 `input_text` 端口（或直接输入）
3. 在 `pattern` 参数输入正则表达式

**智能生成正则**：
1. 点击 `pattern` 右侧的 🪄 按钮
2. 输入多个相似格式的示例文本
3. 用 `[]` 标记需要提取的部分
4. 点击"🪄 智能生成"→"▶ 测试"→"✓ 确认使用"

**示例**：
```
示例文本：
CG 渲染_2026-02-26_21-11-27
彩虹小马_2026-02-26_21-18-55

标记模板：[CG 渲染]_2026-02-26_21-11-27

生成正则：([^_]+)_\d{4}-\d{2}-\d{2}_\d{2}-\d{2}-\d{2}

提取结果：CG 渲染、彩虹小马
```

### 多线程处理节点

**用途**：使用线程池并发处理列表中的每个元素（适合 I/O 密集型任务）

**使用步骤**：
1. 添加"多线程处理"节点
2. 设置参数：
   - **输入列表**：JSON 格式列表（如 `[1, 2, 3, 4, 5]`）
   - **线程数量**：并发线程数（默认 4）
   - **返回顺序**：按输入顺序 / 按完成顺序
3. 连接"迭代值"到并发处理节点
4. 连接"汇总结果"到接收完整结果的节点

**返回顺序**：
- **按输入顺序**：结果与输入顺序一致（推荐）
- **按完成顺序**：结果按线程完成顺序排列

**注意事项**：
- 每个线程在独立子进程中执行，避免线程安全问题
- 单个线程失败不影响其他线程
- 适合 I/O 密集型任务（网络请求、文件读写）

### 添加内置节点

1. 在 `core/nodes/base_nodes.py` 中添加函数：
```python
def 节点名 (参数：类型) -> 返回类型:
    """节点说明文档"""
    return 结果

# 保存源代码字符串（用于序列化）
节点名._source = '''def 节点名 (参数：类型) -> 返回类型:
    """节点说明文档"""
    return 结果
'''
```

2. 在 `core/nodes/node_library.py` 的 `NODE_LIBRARY_CATEGORIZED` 中注册分类

### 执行引擎架构

```
用户点击运行 → graph_executor.topological_sort() 拓扑排序
                    ↓
            batch_executor.BatchGraphExecutor
                    ↓
    ┌───────────────┴───────────────┐
    ↓                               ↓
本地执行（内置节点）        嵌入式执行（自定义节点）
    ↓                               ↓
    └───────────────┬───────────────┘
                    ↓
         输出到 console_stream → 底部控制台
```
