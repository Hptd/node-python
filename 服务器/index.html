👨‍💻 自定义节点示例

用 Python 轻松创建属于你的节点

# 在 base_nodes.py 中定义你的节点函数

def int_add(a: int, b: int) -> int:
    """
    计算两个整数的和。

    参数:
        a: 第一个整数
        b: 第二个整数

    返回:
        两数之和
    """
    return a + b
