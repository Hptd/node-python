📌 插件包介绍

基础数据处理节点包是专为通用数据处理场景设计的节点插件集合，提供文件读写、JSON 处理、字符串操作、数据验证、类型转换、列表处理等基础功能。无需编写代码，即可完成日常数据处理任务。

适用场景

  • ✅ 文件处理：读取/写入文本文件、JSON 文件
  • ✅ 数据清洗：字符串处理、类型转换、空值检测
  • ✅ 数据验证：邮箱、手机号、URL 格式校验
  • ✅ 列表操作：去重、筛选、排序、合并
  • ✅ 任何需要基础数据处理的场景

📥 安装与依赖

安装步骤

  1. 下载 基础数据处理节点包.zip 插件文件

  3. 选择下载的 JSON 文件
  4. 确认后节点将添加到左侧节点库中

第三方依赖

📁 文件读写节点

基础文件操作

文件读取

输入：2 个参数 输出：str

读取文本文件内容。

参数名类型必填默认值说明
file_pathstr是-要读取的文件路径，支持绝对路径和相对路径
encodingstr否"utf-8"文件编码格式，默认为 "utf-8"。常见编码：utf-8, gbk, gb2312, latin-1 等

返回：文件的完整文本内容。如果文件不存在或读取失败，返回空字符串 ""

示例：read_file("data.txt") → "文件内容..."

文件写入

输入：4 个参数 输出：bool

将文本内容写入文件。

参数名类型必填默认值说明
file_pathstr是-要写入的文件路径
contentstr是-要写入文件的文本内容
encodingstr否"utf-8"文件编码格式
modestr否"overwrite"写入模式："overwrite"覆盖写入 / "append"追加写入

返回：写入成功返回 True，失败返回 False

示例：write_file("output.txt", "Hello World") → True

JSON 文件读取

输入：2 个参数 输出：any

读取 JSON 文件并解析为 Python 对象。

参数名类型必填默认值说明
file_pathstr是-要读取的 JSON 文件路径
encodingstr否"utf-8"文件编码格式

返回：解析后的 Python 对象（dict、list、str、int、float 等）。如果读取或解析失败，返回 None

示例：read_json_file("config.json") → {"name": "test", "value": 123}

JSON 文件写入

输入：5 个参数 输出：bool

将 Python 对象写入 JSON 文件。

参数名类型必填默认值说明
file_pathstr是-要写入的 JSON 文件路径
dataany是-要写入的 Python 对象
encodingstr否"utf-8"文件编码格式
indentint否4JSON 缩进空格数，设置为 None 则不格式化（单行输出）
ensure_asciibool否False是否转义非 ASCII 字符：True-转义为 \\uXXXX 格式 / False-保留原始字符

返回：写入成功返回 True，失败返回 False

示例：write_json_file("output.json", {"name": "测试"}) → True

文件是否存在

输入：1 个参数 输出：bool

检查文件或路径是否存在。

参数名类型必填默认值说明
file_pathstr是-要检查的文件或目录路径

返回：如果路径存在返回 True，否则返回 False

示例：file_exists("data.txt") → True

获取文件大小

输入：2 个参数 输出：int

获取文件大小。

参数名类型必填默认值说明
file_pathstr是-要获取大小的文件路径
unitstr否"bytes"返回值的单位："bytes"字节 / "KB"千字节 / "MB"兆字节 / "GB"吉字节

返回：文件大小（根据指定单位返回整数值）。如果文件不存在或读取失败，返回 -1

示例：get_file_size("large_file.zip", unit="MB") → 10

📄 JSON 处理节点

JSON 数据操作

JSON 校验

输入：1 个参数 输出：dict

校验输入字符串是否为合法的 JSON 格式。

参数名类型必填默认值说明
input_stringstr是-要校验的 JSON 格式字符串

返回：包含以下字段的字典：is_valid（是否有效）、error_message（错误信息）、data_type（数据类型）、parsed_data（解析后的数据）

示例：validate_json('{"name": "test"}') → {"is_valid": True, "data_type": "object", ...}

文本转 JSON

输入：2 个参数 输出：dict

尝试将输入文本转换为合法的 JSON 对象。

参数名类型必填默认值说明
input_textstr是-要转换的输入文本
strict_modebool否False是否启用严格模式：True-仅接受标准 JSON / False-尝试多种转换策略

返回：包含以下字段的字典：success（是否成功）、data（转换后的数据）、original_type（原始类型）、conversion_method（转换方法）、error_message（错误信息）

示例：parse_to_json("{'name': 'test'}") → {"success": True, "data": {"name": "test"}, ...}

JSON 格式化

输入：3 个参数 输出：str

将数据格式化为美观的 JSON 字符串。

参数名类型必填默认值说明
input_dataany是-要格式化的数据
indentint否4缩进空格数，设置为 0 或 None 则不格式化（单行输出）
ensure_asciibool否False是否转义非 ASCII 字符：True-转义 / False-保留原始字符

返回：格式化后的 JSON 字符串。如果序列化失败，返回空字符串 ""

示例：format_json({"name": "测试", "value": 123}) → '{\n "name": "测试",\n "value": 123\n}'

JSON 路径提取

输入：2 个参数 输出：any

从 JSON 数据中提取指定路径的值。

参数名类型必填默认值说明
json_dataany是-JSON 数据（dict 或 list）
pathstr是-要提取的路径，支持格式："key"、"key1.key2"、"list[0]"、"list.0"、"data.items.0.name"

返回：提取到的值。如果路径无效或中途失败，返回 None

示例：extract_json_path({"user": {"name": "test"}}, "user.name") → "test"

🔤 字符串处理节点

文本编辑工具

转大写

输入：1 个参数 输出：str

将字符串转换为大写。

参数名类型必填默认值说明
input_stringstr是-要转换的输入字符串

返回：转换后的大写字符串

示例：to_upper("hello world") → "HELLO WORLD"

转小写

输入：1 个参数 输出：str

将字符串转换为小写。

参数名类型必填默认值说明
input_stringstr是-要转换的输入字符串

返回：转换后的小写字符串

示例：to_lower("HELLO WORLD") → "hello world"

去除空白

输入：2 个参数 输出：str

去除字符串空白字符。

参数名类型必填默认值说明
input_stringstr是-要处理的输入字符串
modestr否"both"去除模式："both"两端 / "left"左侧 / "right"右侧

返回：去除空白后的字符串

示例：strip_string(" hello ", mode="left") → "hello "

字符串分割

输入：3 个参数 输出：list

按分隔符分割字符串。

参数名类型必填默认值说明
input_stringstr是-要分割的输入字符串
separatorstr否","分隔符
max_splitint否-1最大分割次数：-1-不限制 / 0 或正整数 - 最多分割指定次数

返回：分割后的字符串列表

示例：split_string("a,b,c,d", separator=",", max_split=2) → ["a", "b", "c,d"]

字符串合并

输入：2 个参数 输出：str

将字符串列表合并为单个字符串。

参数名类型必填默认值说明
input_listlist是-要合并的列表
separatorstr否","连接用的分隔符

返回：合并后的字符串。如果处理失败，返回空字符串 ""

示例：join_strings(["a", "b", "c"], separator="-") → "a-b-c"

字符串替换

输入：4 个参数 输出：str

替换字符串中的子串。

参数名类型必填默认值说明
input_stringstr是-原始输入字符串
oldstr是-要替换的子串（被替换的内容）
newstr是-新的子串（替换后的内容）
countint否-1最大替换次数：-1-替换所有 / 0 或正整数 - 最多替换指定次数

返回：替换后的新字符串

示例：replace_string("aaa", "a", "b", count=2) → "bba"

字符串截取

输入：3 个参数 输出：str

截取字符串子串。

参数名类型必填默认值说明
input_stringstr是-原始输入字符串
startint是-起始索引（从 0 开始）。正数 - 从左往右数 / 负数 - 从右往左数
endint否None结束索引（不包含该位置）。None-截取到字符串末尾

返回：截取后的子串

示例：substring("hello world", 0, 5) → "hello"

✅ 数据验证节点

格式校验工具

邮箱校验

输入：1 个参数 输出：dict

校验字符串是否为有效的邮箱地址格式。

参数名类型必填默认值说明
input_stringstr是-要校验的邮箱地址字符串

返回：包含以下字段的字典：is_valid（是否有效）、message（校验结果信息）、domain（邮箱域名）

示例：validate_email("test@example.com") → {"is_valid": True, "message": "Valid", "domain": "example.com"}

手机号校验

输入：2 个参数 输出：dict

校验字符串是否为有效的手机号码格式。

参数名类型必填默认值说明
input_stringstr是-要校验的手机号码字符串
country_codestr否"CN"国家/地区代码："CN"-中国大陆

返回：包含以下字段的字典：is_valid（是否有效）、message（校验结果信息）、formatted（格式化后的手机号）

示例：validate_phone("13812345678") → {"is_valid": True, "message": "Valid", "formatted": "13812345678"}

URL 校验

输入：1 个参数 输出：dict

校验字符串是否为有效的 URL 格式。

参数名类型必填默认值说明
input_stringstr是-要校验的 URL 字符串

返回：包含以下字段的字典：is_valid（是否有效）、message（校验结果信息）、scheme（协议）、domain（域名）

示例：validate_url("https://www.example.com/path") → {"is_valid": True, "scheme": "https", "domain": "www.example.com"}

空值检测

输入：1 个参数 输出：bool

检测值是否为空。

参数名类型必填默认值说明
input_valueany是-要检测的任意类型值

返回：如果值为空返回 True，否则返回 False

示例：is_empty("") → True

数字检测

输入：1 个参数 输出：bool

检测值是否为数字或可转换为数字的字符串。

参数名类型必填默认值说明
input_valueany是-要检测的任意类型值

返回：如果是数字或可转换为数字返回 True，否则返回 False

示例：is_numeric("123") → True

📋 列表处理节点

列表数据操作

列表去重

输入：2 个参数 输出：list

去除列表中的重复元素。

参数名类型必填默认值说明
input_listlist是-要去重的列表。如果不是列表类型，返回空列表
keep_orderbool否True是否保持原始顺序：True-保持顺序（遍历去重） / False-不保证顺序（set 去重，更快）

返回：去重后的新列表

示例：list_unique([1, 2, 2, 3, 3, 3]) → [1, 2, 3]

列表筛选

输入：2 个参数 输出：list

筛选列表中的元素。

参数名类型必填默认值说明
input_listlist是-要筛选的列表。如果不是列表类型，返回空列表
filter_typestr否"not_empty"筛选类型："not_empty"去除空值 / "string_only"仅保留字符串 / "number_only"仅保留数字 / "dict_only"仅保留字典

返回：筛选后的新列表

示例：list_filter([1, "a", 2.5, {}], filter_type="number_only") → [1, 2.5]

列表排序

输入：3 个参数 输出：list

对列表进行排序。

参数名类型必填默认值说明
input_listlist是-要排序的列表。如果不是列表类型，返回空列表
orderstr否"asc"排序顺序："asc"升序 / "desc"降序
keystr否None当列表元素为字典时，指定按哪个键排序

返回：排序后的新列表。如果排序失败，返回原列表

示例：list_sort([3, 1, 4], order="desc") → [4, 3, 1]

列表合并

输入：3 个参数 输出：list

合并两个列表。

参数名类型必填默认值说明
list1list是-第一个待合并的列表。如果不是列表，转换为空列表
list2list是-第二个待合并的列表。如果不是列表，转换为空列表
remove_duplicatesbool否False是否去除重复项：True-去重后返回 / False-直接合并

返回：合并后的新列表，根据参数决定是否去重

示例：list_merge([1, 2], [2, 3], remove_duplicates=True) → [1, 2, 3]

列表长度

输入：1 个参数 输出：int

获取列表长度。

参数名类型必填默认值说明
input_listany是-要获取长度的对象。支持：list、tuple、set、str、dict，其他类型返回 0

返回：对象的元素数量。如果类型不支持，返回 0

示例：list_length([1, 2, 3]) → 3
